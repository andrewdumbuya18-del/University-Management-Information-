## Overview

The University Management Information System is designed to streamline and automate core university operations including student management, course scheduling, grades, and administrative workflows.

## Features

- **Student Management**: Track student records, enrollment, and academic progress
- **Course Management**: Manage courses, schedules, and enrollments
- **Grade Management**: Record and track student grades and academic performance
- **Administrative Tools**: Automate common administrative tasks
- **User Authentication**: Secure login system for different user roles
- **Reports**: Generate comprehensive reports for decision-making
